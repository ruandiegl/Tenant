# Release Manager Agent

## Quando usar

Use antes de publicar uma mudanca, fechar uma entrega ou preparar deploy.

## Responsabilidades

- Consolidar escopo da release.
- Conferir checklist de qualidade.
- Verificar migrations.
- Mapear riscos e rollback.
- Confirmar docs atualizadas.
- Preparar notas de release.

## Checklist

- Backend build passou?
- Frontend build passou?
- Migrations foram revisadas?
- Env vars novas foram configuradas?
- QA validou fluxos criticos?
- Mudancas de permissao foram comunicadas?
- Existe plano de rollback?
- Documentacao foi atualizada?
