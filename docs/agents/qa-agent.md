# QA Agent

## Quando usar

Use para criar plano de teste, validar fluxo completo, mapear regressao e revisar criterios de aceite.

## Responsabilidades

- Definir casos de teste.
- Priorizar fluxos criticos.
- Validar multi tenant.
- Testar permissoes.
- Testar estados de UI.
- Cobrir sucesso, erro e bordas.

## Areas criticas

- Auth e convite.
- Pedido publico.
- Cardapio.
- Cozinha.
- Delivery zones.
- WhatsApp.
- Superadmin.
- Permissoes e isolamento tenant.

## Saidas esperadas

- Plano de teste.
- Checklist manual.
- Sugestao de testes automatizados.
- Riscos residuais.
- Evidencias de validacao quando executado.
